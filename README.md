# مرجع مطوّر لموقع مركز الأمجاد للغات والتدريب

هذا الملف يشرح محتويات الموقع الموجود في المجلد. الهدف هو أن يكون مرجعًا لأي مطوّر يعمل على المشروع.

## الملفات الموجودة

- `index.html` : الصفحة الرئيسية للموقع.
- `logo.png` : صورة الشعار المستخدمة في الصفحة.
- `logo1 (2).png` : صورة إضافية موجودة في المجلد (غير مستخدمة في `index.html`).

## محتوى `index.html`

الملف هو صفحة ويب HTML بسيطة تحتوي على التالي:

1. `<!DOCTYPE html>` ووسوم HTML الأساسية.
2. الوسم `<html lang="ar" dir="rtl">` لعرض المحتوى باللغة العربية وتفعيل الاتجاه من اليمين إلى اليسار.
3. قسم `<head>` يحتوي على:
   - ترميز UTF-8.
   - إعدادات عرض الهواتف المحمولة عبر `viewport`.
   - عنوان الصفحة `مركز الأمجاد للغات والتدريب`.
   - شيفرة CSS داخلية داخل وسم `<style>`.

### أقسام الصفحة

- `header`:
  - يحتوي على صورة الشعار `logo.png`.
  - يحتوي على عنوان الموقع `مركز الأمجاد للغات والتدريب` ونص فرعي.
  - تم تعديل التصميم ليعرض الشعار بجانب الاسم في الجهة اليمنى، بخلفية شفافة.

- `section.cards`:
  - تعرض مجموعة بطاقات روابط خارجية.
  - كل بطاقة هي `<a class="card">` تحتوي على أيقونة، عنوان فرعي، نص، وزر.

- البطاقات الموجودة حاليًا:
  - إنستغرام: يفتح حساب الإنستغرام.
  - فيسبوك: يفتح الصفحة الرسمية على فيسبوك.
  - واتساب مباشر: يفتح محادثة واتساب برقم مباشر.
  - قناة واتساب: يفتح قناة المركز على واتساب.
  - الموقع: يفتح موقع المركز على Google Maps.

- `footer`:
  - يعرض نص الحقوق: `جميع الحقوق محفوظة © 2026 | تصميم وتطوير: يحيى المريسي`.

## تنسيق CSS الموجود

الصفحة تستخدم CSS داخلية لتنسيق العناصر التالية:

- إعداد المتغيرات اللونية في `:root` مثل الأزرق، الأصفر، الأبيض، الرمادي.
- إعادة تهيئة الهوامش والبادينغ لكل العناصر.
- تصميم خلفية الصفحة بلون رمادي فاتح.
- تنسيق الهيدر بخلفية زرقاء ونص أبيض.
- تنسيق بطاقات الروابط بمظهر بطاقات بيضاء مع ظل وأزرار صفراء.
- دعم واجهة متجاوبة عبر وسائط `@media (max-width: 600px)`.

## كيفية التشغيل

يمكن فتح الموقع بطريقتين:

1. فتح الملف مباشرة:
   - انقر مزدوجًا على `index.html` من مستعرض الملفات.
   - سيتم فتحه في المتصفح الافتراضي.

2. تشغيله من خلال سطر الأوامر في PowerShell:
   - افتح PowerShell.
   - انتقل إلى المجلد:
     ```powershell
     Set-Location -Path "G:\مركز الامجاد"
     ```
   - ثم افتح الصفحة:
     ```powershell
     Invoke-Item .\index.html
     ```

## ملاحظات للمطوّر

- تأكد أن اسم ملف الشعار هو `logo.png` لأن `index.html` يشير إليه بهذا الاسم.
- إذا أردت إضافة محتوى جديد، يمكنك تعديل `index.html` مباشرة أو نقل CSS إلى ملف خارجي إذا كان المشروع سيكبر.
- يمكن إضافة صفحة `styles.css` و `script.js` لاحقًا إذا رغبت في فصل المظهر عن المحتوى.
- إذا كان هناك روابط إضافية أو صفحات داخلية، يمكن توسيع قسم البطاقات في `section.cards`.

## تحسينات مقترحة لاحقًا

- إضافة قسم "عن المركز" أو "خدماتنا".
- استخدام `target="_blank" rel="noopener noreferrer"` للروابط الخارجية لزيادة الأمان.
- إضافة أيقونات من مكتبة خطوط مثل Font Awesome لتحسين الشكل.
- إضافة خريطة مدمجة `iframe` بدلاً من رابط فقط.
